ABOUT Apache Spark Performance

Our goal is to work with the Apache Spark community to further enhance the scalability and reliability of the Apache Spark. The data available on this site allows community members and potential Spark customers to closely track performance trend of the Apache Spark. Ultimately, we hope that this project will help community to fix performance regression issue quickly, thus providing better Apache spark code to end customers.

Currently, the firstly adopted benchmarks include HiBench (a benchmark suite to evaluate big data framework like Hadoop MR, Spark from Intel®), Spark-perf (a performance testing framework for Apache Spark from Databricks®). Additional benchmarks will be added as they become available.
